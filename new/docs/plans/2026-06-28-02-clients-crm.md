# Agency OS — Plan 02: Client Accounts & CRM Deals Pipeline Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the CRM deals pipeline (PRD §5.3) and Client Accounts hub (PRD §5.4) on top of the Plan 01 foundation. Deliver: a tenant-scoped data model for clients, contacts, services/retainers, deals, configurable deal stages, and deal activity; a drag-and-drop Kanban deal board (dnd-kit) with persisted ordering, configurable stages, and stage-change timestamps for velocity reporting; pipeline value rollups (weighted value, value by stage, win rate); a one-click "Closed Won → create client" conversion that never double-enters; and a clients accounts list plus a client detail hub shell. Every new tenant-scoped table gets an RLS isolation test using the Plan 01 harness.

**Architecture:** All CRM and client data lives in the agency's data plane and is **staff-only** (PRD §3.3: CRM/deals and Client accounts are Founder/Team only; clients have no access). We still enforce tenancy with RLS so the rules compose with later client-portal plans: `deal`, `deal_stage`, `deal_activity`, and `contact` belong to the **agency org** (their `organization_id` is the agency organization id) and are readable/writable only by agency staff via `public.is_agency_staff()`. `client` rows are the per-client *organization profile*: each `client` row carries `organization_id` = the client's own `organizations.id` (the client-type org created in Plan 01), so client-scoped data added by later plans (analytics, invoices, tasks) keys off the same `organization_id`. `client_contact` and `service` are scoped by the client's `organization_id`. Reads/writes route through the anon Supabase client under RLS (defense in depth: server actions also scope every query by org). `service_role` is used only by the seed/admin, never in user-facing queries — matching Plan 01.

The Kanban board uses **dnd-kit** with optimistic UI and server reconciliation: each deal carries a fractional `position` (double precision) within its `deal_stage`; dragging persists `(stageId, position)` via a server action; stage transitions append a `deal_activity` row of type `stage_change` recording `from_stage_id`/`to_stage_id` + timestamp for velocity reporting. Pipeline rollups are computed in SQL (weighted value = Σ `value × stage.probability`).

**Tech Stack:** Next.js 16 (App Router, TS strict) · Drizzle ORM + drizzle-kit · postgres.js · Supabase Postgres + Auth (Plan 01 helpers `public.has_org_access(uuid)`, `public.is_agency_staff()`) · Tailwind + shadcn/ui · **dnd-kit** (`@dnd-kit/core`, `@dnd-kit/sortable`, `@dnd-kit/utilities`) · Vitest (RLS + unit tests via the Plan 01 `tests/helpers/db.ts` harness).

**Depends on:** Plan 01 (foundation) — `organizations`/`profiles`/`memberships` tables, `org_type`/`app_role` enums, `public.has_org_access()`/`public.is_agency_staff()`, `custom_access_token_hook`, `scripts/seed.ts`, and `tests/helpers/db.ts` (`asUser()`, `userIdByEmail()`). Do **not** redefine any of those.

```sql
-- Enable RLS on every new table.
alter table public.client          enable row level security;
alter table public.client_contact  enable row level security;
alter table public.service         enable row level security;
alter table public.contact         enable row level security;

alter table public.deal_activity   enable row level security;

--    writes are agency-staff-only. ──

create policy client_select on public.client
  for select using (public.has_org_access(organization_id));
create policy client_write on public.client

create policy client_contact_select on public.client_contact
  for select using (public.has_org_access(organization_id));
create policy client_contact_write on public.client_contact
  for all using (public.is_agency_staff()) with check (public.is_agency_staff());

create policy service_select on public.service

create policy service_write on public.service

create policy contact_all on public.contact
  for all using (public.is_agency_staff()) with check (public.is_agency_staff());

create policy deal_stage_all on public.deal_stage
  for all using (public.is_agency_staff()) with check (public.is_agency_staff());

create policy deal_all on public.deal
  for all using (public.is_agency_staff()) with check (public.is_agency_staff());

create policy deal_activity_all on public.deal_activity
  for all using (public.is_agency_staff()) with check (public.is_agency_staff());

-- ── Seed the default pipeline stages for the agency org (PRD §5.3 defaults). ──
-- Idempotent: only insert if the agency org has no stages yet.
do $$
declare
  v_agency uuid;
begin
  select id into v_agency from public.organizations where type = 'agency' order by created_at limit 1;
  if v_agency is null then

  end if;
  if not exists (select 1 from public.deal_stage where organization_id = v_agency) then
    insert into public.deal_stage (organization_id, name, position, probability, is_won, is_lost) values
      (v_agency, 'Lead',          0, 0.100, false, false),
      (v_agency, 'Proposal Sent', 1, 0.400, false, false),
      (v_agency, 'Negotiation',   2, 0.700, false, false),
      (v_agency, 'Closed Won',    3, 1.000, true,  false),
```

## Task 4: CRM/client demo seed (used by the RLS + conversion tests)

**Files:**
- Create: `scripts/seed-crm.ts`

This builds on the Plan 01 seed (which created the agency org, two client orgs, and users). It adds: a CRM prospect contact, one deal in the "Lead" stage for the agency, and one existing `client` profile mapped to the `client-one` org. Idempotent like the Plan 01 seed.

- [ ] **Step 1: Write `scripts/seed-crm.ts`**

```ts
import 'dotenv/config'
import { drizzle } from 'drizzle-orm/postgres-js'
import postgres from 'postgres'
import { and, eq } from 'drizzle-orm'
import * as schema from '../src/db/schema'

const db = drizzle(postgres(process.env.DATABASE_URL!, { prepare: false }), { schema })

async function orgIdBySlug(slug: string): Promise<string> {
  const rows = await db
    .select()
    .from(schema.organizations)
    .where(eq(schema.organizations.slug, slug))
  if (!rows[0]) throw new Error(`org not found: ${slug} (run pnpm db:seed first)`)
  return rows[0].id
}

async function stageIdByName(orgId: string, name: string): Promise<string> {
  const rows = await db
    .select()
    .from(schema.dealStages)
    .where(and(eq(schema.dealStages.organizationId, orgId), eq(schema.dealStages.name, name)))
  if (!rows[0]) throw new Error(`stage not found: ${name} (run pnpm db:migrate first)`)
  return rows[0].id
}

async function main() {
  const agency = await orgIdBySlug('milktree')
  const clientOneOrg = await orgIdBySlug('client-one')
  const leadStage = await stageIdByName(agency, 'Lead')

  // CRM prospect contact (agency-owned).
  const existingContact = await db
    .select()
    .from(schema.contacts)
    .where(and(eq(schema.contacts.organizationId, agency), eq(schema.contacts.name, 'Dana Prospect')))
  const contactId =
    existingContact[0]?.id ??
    (
      await db
        .insert(schema.contacts)
        .values({
          organizationId: agency,
          name: 'Dana Prospect',
          email: 'dana@prospect.example',
          company: 'Prospect Co',
        })
        .returning()
    )[0]!.id

  // A deal in Lead (agency-owned).
  const existingDeal = await db
    .select()
    .from(schema.deals)
    .where(and(eq(schema.deals.organizationId, agency), eq(schema.deals.title, 'Prospect Co — SEO retainer')))
  if (!existingDeal[0]) {
    await db.insert(schema.deals).values({
      organizationId: agency,
      stageId: leadStage,
      title: 'Prospect Co — SEO retainer',
      companyName: 'Prospect Co',
      contactId,
      services: ['SEO'],
      value: 250000, // £2,500.00 in pence
      source: 'inbound',
      position: 1000,
    })
  }

  // An existing client profile mapped to the client-one org.
  const existingClient = await db
    .select()
    .from(schema.clients)
    .where(eq(schema.clients.organizationId, clientOneOrg))
  if (!existingClient[0]) {
    await db.insert(schema.clients).values({
      organizationId: clientOneOrg,
      name: 'Client One Ltd',
      services: ['SEO', 'Google Ads'],
      health: 'good',
    })
  }

  console.log('CRM seed complete')
  process.exit(0)
}

main().catch((e) => {
  console.error(e)
  process.exit(1)
})
```

```ts
// Fractional ordering for Kanban cards. A new card's position is computed
// between its neighbours so reordering only ever updates one row.
export const STEP = 1000

/**
 * Compute a position for a card dropped between `before` and `after`.
 * - before === null  → dropped at the top of the column.
 * - after  === null  → dropped at the bottom of the column.
 * - both null        → empty column.
 */
export function positionBetween(before: number | null, after: number | null): number {
  if (before === null && after === null) return STEP
  if (before === null) return after! / 2
  if (after === null) return before + STEP
  return (before + after) / 2
}
```

```ts
export type RollupStage = {
  id: string
  name: string
  probability: number
  isWon: boolean
  isLost: boolean
}

export type RollupDeal = {
  id: string
  stageId: string
  value: number
}

/** Sum of deal values per stage id. Every passed-in stage gets a key (0 if empty). */
export function valueByStage(stages: RollupStage[], deals: RollupDeal[]): Record<string, number> {
  const totals: Record<string, number> = {}
  for (const s of stages) totals[s.id] = 0
  for (const d of deals) {
    totals[d.stageId] = (totals[d.stageId] ?? 0) + d.value
  }
  return totals
}

/** Σ value × stage.probability across deals in non-terminal stages. */
export function weightedPipelineValue(stages: RollupStage[], deals: RollupDeal[]): number {
  const byId = new Map(stages.map((s) => [s.id, s]))
  let total = 0
  for (const d of deals) {
    const stage = byId.get(d.stageId)
    if (!stage || stage.isWon || stage.isLost) continue
    total += d.value * stage.probability
  }
  return total
}

/** won / (won + lost) across deals in terminal stages; 0 when none. */
export function winRate(stages: RollupStage[], deals: RollupDeal[]): number {
  const byId = new Map(stages.map((s) => [s.id, s]))
  let won = 0
  let lost = 0
  for (const d of deals) {
    const stage = byId.get(d.stageId)
    if (!stage) continue
    if (stage.isWon) won += 1
    else if (stage.isLost) lost += 1
  }
  const terminal = won + lost
  return terminal === 0 ? 0 : won / terminal
}
```

```ts
  const stageRows = await db
    .select()
    .from(dealStages)
    .where(eq(dealStages.organizationId, orgId))
    .orderBy(asc(dealStages.position))

    .orderBy(asc(deals.stageId), asc(deals.position))

    id: d.id,
    stageId: d.stageId,
    value: d.value,

  const total = Object.values(byStage).reduce((a, b) => a + b, 0)
```

```ts
/** Monthly-recurring revenue in pence: active retainer services normalised to monthly. */
function monthlyAmount(s: Service): number {
  if (!s.isActive || s.kind !== 'retainer') return 0
  switch (s.interval) {
    case 'monthly':
      return s.amount
    case 'quarterly':
      return Math.round(s.amount / 3)
    case 'annual':
      return Math.round(s.amount / 12)
    default:
      return s.amount
  }
}

export async function listClients(orgId: string): Promise<ClientListRow[]> {
  const clientRows = await db
    .select()
    .from(clients)
    .where(eq(clients.organizationId, orgId))
    .orderBy(asc(clients.name))

  const serviceRows = await db.select().from(services).where(eq(services.organizationId, orgId))

  const byClient = new Map<string, Service[]>()
  for (const s of serviceRows) {
    const arr = byClient.get(s.clientId) ?? []
    arr.push(s)
    byClient.set(s.clientId, arr)
  }

  return clientRows.map((c) => {
    const svc = byClient.get(c.id) ?? []
    return {
      ...c,
      mrr: svc.reduce((sum, s) => sum + monthlyAmount(s), 0),
      openServices: svc.filter((s) => s.isActive).length,

  })
```

```ts
  orgId: string
  stageId: string
  title: string
  companyName: string
  value?: number
  source?: 'referral' | 'inbound' | 'outbound' | 'event' | 'partner' | 'other'
  actorId?: string
}

export async function createDeal(input: CreateDealInput): Promise<{ id: string }> {
  return db.transaction(async (tx) => {
    // place at the bottom of the stage.
    const [last] = await tx
      .select({ position: deals.position })
      .from(deals)
      .where(and(eq(deals.organizationId, input.orgId), eq(deals.stageId, input.stageId)))
      .orderBy(desc(deals.position))
      .limit(1)
    const position = positionBetween(last?.position ?? null, null)

    const [row] = await tx
      .insert(deals)
      .values({
        organizationId: input.orgId,
        stageId: input.stageId,
        title: input.title,
        companyName: input.companyName,
        value: input.value ?? 0,
        source: input.source ?? 'inbound',
        position,
      })
      .returning({ id: deals.id })

    await tx.insert(dealActivities).values({
      organizationId: input.orgId,
      dealId: row!.id,
      type: 'created',
      actorId: input.actorId ?? null,
    })
    return { id: row!.id }
  })
}

export type MoveDealInput = {
  orgId: string
  dealId: string
  toStageId: string
  beforeId: string | null // neighbour above the drop target (or null = top)
  afterId: string | null // neighbour below the drop target (or null = bottom)
  actorId?: string
}

export async function moveDeal(input: MoveDealInput): Promise<void> {
  await db.transaction(async (tx) => {
    const [deal] = await tx
      .select()
      .from(deals)
      .where(and(eq(deals.organizationId, input.orgId), eq(deals.id, input.dealId)))
    if (!deal) throw new Error('deal not found')

    const beforePos = input.beforeId
      ? (await tx.select({ p: deals.position }).from(deals).where(eq(deals.id, input.beforeId)))[0]?.p ?? null
      : null
    const afterPos = input.afterId
      ? (await tx.select({ p: deals.position }).from(deals).where(eq(deals.id, input.afterId)))[0]?.p ?? null
      : null

    const newPosition = positionBetween(beforePos, afterPos)
    const stageChanged = deal.stageId !== input.toStageId

    await tx
      .update(deals)
      .set({ stageId: input.toStageId, position: newPosition, updatedAt: new Date() })
      .where(eq(deals.id, input.dealId))

    if (stageChanged) {
      await tx.insert(dealActivities).values({
        organizationId: input.orgId,
        dealId: input.dealId,
        type: 'stage_change',
        fromStageId: deal.stageId,
        toStageId: input.toStageId,
        actorId: input.actorId ?? null,
      })
    }
  })
}

export type ConvertResult = { clientId: string; created: boolean }

/**
 * Convert a Closed-Won deal into a client organization profile.
 * Idempotent: if the deal already has convertedClientId, returns it (created:false).
 * Requires a client-type organization id to attach the profile to.
 */
export async function convertWonDeal(input: {
  orgId: string
  dealId: string
  clientOrganizationId: string
  actorId?: string
}): Promise<ConvertResult> {
  return db.transaction(async (tx) => {
    const [deal] = await tx
      .select()
      .from(deals)
      .where(and(eq(deals.organizationId, input.orgId), eq(deals.id, input.dealId)))
    if (!deal) throw new Error('deal not found')

    // idempotency guard — never double-create.
    if (deal.convertedClientId) {
      return { clientId: deal.convertedClientId, created: false }
    }

    const [stage] = await tx.select().from(dealStages).where(eq(dealStages.id, deal.stageId))
    if (!stage?.isWon) throw new Error('deal is not in a Closed Won stage')

    // if a client profile already exists for the target org, reuse it.
    const [existing] = await tx
      .select()
      .from(clients)
      .where(eq(clients.organizationId, input.clientOrganizationId))

    let clientId: string
    let created = false
    if (existing) {
      clientId = existing.id
    } else {
      const [row] = await tx
        .insert(clients)
        .values({
          organizationId: input.clientOrganizationId,
          name: deal.companyName,
          services: deal.services,
          sourceDealId: deal.id,
          health: 'good',
        })
        .returning({ id: clients.id })
      clientId = row!.id
      created = true
    }

    await tx.update(deals).set({ convertedClientId: clientId }).where(eq(deals.id, deal.id))

    await tx.insert(dealActivities).values({
      organizationId: input.orgId,
      dealId: deal.id,
      type: 'converted',
      actorId: input.actorId ?? null,
      note: `Converted to client ${clientId}`,
    })

    return { clientId, created }
  })
}
```

- [ ] **Step 3: Board `src/components/crm/deal-board.tsx`**

```tsx
'use client'
import { useState } from 'react'
import {
  DndContext,
  PointerSensor,
  useSensor,
  useSensors,
  type DragEndEvent,
} from '@dnd-kit/core'
import type { Deal, DealStage } from '@/db/types'
import { DealColumn } from './deal-column'

export type MoveDealAction = (input: {
  dealId: string
  toStageId: string
  beforeId: string | null
  afterId: string | null
}) => Promise<void>

export function DealBoard({
  stages,
  initialDealsByStage,
  byStageTotals,
  moveDealAction,
}: {
  stages: DealStage[]
  initialDealsByStage: Record<string, Deal[]>
  byStageTotals: Record<string, number>
  moveDealAction: MoveDealAction
}) {
  const [dealsByStage, setDealsByStage] = useState(initialDealsByStage)
  const sensors = useSensors(useSensor(PointerSensor, { activationConstraint: { distance: 4 } }))

  function findStageOf(dealId: string): string | null {
    for (const [stageId, list] of Object.entries(dealsByStage)) {
      if (list.some((d) => d.id === dealId)) return stageId
    }
    return null
  }

  async function onDragEnd(event: DragEndEvent) {
    const { active, over } = event
    if (!over) return
    const dealId = String(active.id)
    const fromStage = findStageOf(dealId)
    if (!fromStage) return

    // `over.id` is either a stage id (empty column) or a deal id (a card).
    const overId = String(over.id)
    const toStage = stages.some((s) => s.id === overId) ? overId : findStageOf(overId)
    if (!toStage) return

    const moving = dealsByStage[fromStage]!.find((d) => d.id === dealId)!
    const target = [...dealsByStage[toStage]!.filter((d) => d.id !== dealId)]
    const overIndex = target.findIndex((d) => d.id === overId)
    const insertAt = overIndex === -1 ? target.length : overIndex
    target.splice(insertAt, 0, { ...moving, stageId: toStage })

    const beforeId = insertAt > 0 ? target[insertAt - 1]!.id : null
    const afterId = insertAt < target.length - 1 ? target[insertAt + 1]!.id : null

    // optimistic update.
    const prev = dealsByStage
    setDealsByStage({
      ...dealsByStage,
      [fromStage]: dealsByStage[fromStage]!.filter((d) => d.id !== dealId),
      [toStage]: target,
    })

    try {
      await moveDealAction({ dealId, toStageId: toStage, beforeId, afterId })
    } catch {
      setDealsByStage(prev) // reconcile on failure.
    }
  }

  return (
    <DndContext sensors={sensors} onDragEnd={onDragEnd}>
      <div className="flex gap-3 overflow-x-auto pb-4">
        {stages.map((s) => (
          <DealColumn
            key={s.id}
            stage={s}
            deals={dealsByStage[s.id] ?? []}
            total={byStageTotals[s.id] ?? 0}
          />
        ))}
      </div>
    </DndContext>
  )
}
```
