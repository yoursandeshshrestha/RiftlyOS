# Agency OS — Plan 03: Task Management & Time-Entry Engine Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the Task Management module (Kanban board with configurable columns, dnd-kit drag/reorder with server-persisted positions, full task fields, comments, attachments, list + My Tasks views) and the time-entry engine (start/stop timer + manual entry with a billable flag, attached to tasks) — with two-way client collaboration (a client can create and comment on tasks in their own org board but cannot reassign or delete agency tasks; client-created tasks are visually distinguished) and Postgres RLS proving a client only ever sees their own board.

**Architecture:** Six new tenant-scoped tables — `boards`, `board_columns`, `tasks`, `task_comments`, `task_attachments`, `time_entries` — every one carrying `organization_id` as the leading column of a composite index, with RLS enabled and policies that reuse the Plan 01 helpers `public.has_org_access(uuid)` and `public.is_agency_staff()`. A new `task_origin` enum (`agency | client`) marks who created a task and drives both the write-policy split (clients may insert/comment but never reassign/delete agency-origin rows) and the UI visual distinction. Drag-and-drop reordering is optimistic on the client (dnd-kit) and reconciled by a transactional Server Action that rewrites `position` for the affected column(s). Timers persist a `started_at` row so they survive navigation/refresh; stopping computes `minutes`. All mutations go through Server Actions backed by the RLS-scoped Supabase server client (defense in depth), never `service_role`.

**Tech Stack:** Next.js 16 (App Router, TypeScript strict) · Drizzle ORM + drizzle-kit · postgres.js · Supabase (Postgres + Auth + RLS) · `@dnd-kit/core` + `@dnd-kit/sortable` · Tailwind + shadcn/ui · Vitest (RLS + logic tests).

**Dependencies (assume already built; do NOT re-spec):**
- **Plan 01 — Foundation:** `org_type` & `app_role` enums; `organizations`, `profiles`, `memberships` tables; `public.has_org_access(uuid)` and `public.is_agency_staff()` SECURITY DEFINER functions; `custom_access_token_hook`; `scripts/seed.ts`; the Vitest harness `tests/helpers/db.ts` exporting `sql`, `asUser(userId, fn)`, and `userIdByEmail(email)`; the role-based app shells under `src/app/(internal)/` and `src/app/(portal)/`; `src/lib/auth.ts` exporting `getSession()` and `isStaff()`; the Drizzle client `src/db/index.ts`.
- **Plan 02 — Client:** the `clients` table (one row per client organization, `id` = the client `organizations.id`) and client-account UI. This plan references a client's organization via `organization_id` on the agency/client org model from Plan 01; it does not redefine the client table.

**Prerequisites:** local Supabase running (`pnpm dlx supabase start`), Plan 01 + Plan 02 migrations applied, and `pnpm db:seed` run so the seed users exist.

```ts
// ── Task Management & Time (Plan 03) ─────────────────────────────────────────

// Who created the task: agency staff or the client. Drives the two-way write
// rules and the visual distinction on the board.
export const taskOrigin = pgEnum('task_origin', ['agency', 'client'])

export const taskPriority = pgEnum('task_priority', ['low', 'medium', 'high', 'urgent'])

// timer = created/closed by start/stop; manual = hand-entered.
export const timeEntrySource = pgEnum('time_entry_source', ['timer', 'manual'])

// A board belongs to a client organization (the client's board) or to the
// agency org (the internal "agency ops" board). organization_id is the tenant.
export const boards = pgTable(
  'boards',
  {
    id: uuid('id').defaultRandom().primaryKey(),
    organizationId: uuid('organization_id')
      .notNull()
      .references(() => organizations.id, { onDelete: 'cascade' }),
    name: text('name').notNull(),
    isOps: boolean('is_ops').notNull().default(false),
    createdAt: timestamp('created_at', { withTimezone: true }).defaultNow().notNull(),
  },
  (t) => ({
    // tenant-leading composite index (RLS performance rule, §9).
    orgIdx: index('idx_boards_org').on(t.organizationId, t.id),
  }),
)

export const boardColumns = pgTable(
  'board_columns',
  {
    id: uuid('id').defaultRandom().primaryKey(),
    organizationId: uuid('organization_id')
      .notNull()
      .references(() => organizations.id, { onDelete: 'cascade' }),
    boardId: uuid('board_id')
      .notNull()
      .references(() => boards.id, { onDelete: 'cascade' }),
    name: text('name').notNull(),
    position: integer('position').notNull(),
    // The intake column receives client-created tasks by default.
    isIntake: boolean('is_intake').notNull().default(false),
    createdAt: timestamp('created_at', { withTimezone: true }).defaultNow().notNull(),
  },
  (t) => ({
    orgIdx: index('idx_board_columns_org').on(t.organizationId, t.boardId, t.position),
  }),
)

export const tasks = pgTable(
  'tasks',
  {
    id: uuid('id').defaultRandom().primaryKey(),
    organizationId: uuid('organization_id')
      .notNull()
      .references(() => organizations.id, { onDelete: 'cascade' }),
    boardId: uuid('board_id')
      .notNull()
      .references(() => boards.id, { onDelete: 'cascade' }),
    columnId: uuid('column_id')
      .notNull()
      .references(() => boardColumns.id, { onDelete: 'restrict' }),
    title: text('title').notNull(),
    description: text('description'),
    assigneeId: uuid('assignee_id').references(() => profiles.id, { onDelete: 'set null' }),
    dueDate: timestamp('due_date', { withTimezone: true }),
    priority: taskPriority('priority').notNull().default('medium'),
    // Ordering within a column. Recomputed on drag-reorder.
    position: integer('position').notNull(),
    origin: taskOrigin('origin').notNull().default('agency'),
    createdBy: uuid('created_by')
      .notNull()
      .references(() => profiles.id, { onDelete: 'cascade' }),
    createdAt: timestamp('created_at', { withTimezone: true }).defaultNow().notNull(),
    updatedAt: timestamp('updated_at', { withTimezone: true }).defaultNow().notNull(),
  },
  (t) => ({
    orgIdx: index('idx_tasks_org').on(t.organizationId, t.boardId, t.columnId, t.position),
    assigneeIdx: index('idx_tasks_assignee').on(t.organizationId, t.assigneeId),
  }),
)

export const taskComments = pgTable(
  'task_comments',
  {
    id: uuid('id').defaultRandom().primaryKey(),
    organizationId: uuid('organization_id')
      .notNull()
      .references(() => organizations.id, { onDelete: 'cascade' }),
    taskId: uuid('task_id')
      .notNull()
      .references(() => tasks.id, { onDelete: 'cascade' }),
    authorId: uuid('author_id')
      .notNull()
      .references(() => profiles.id, { onDelete: 'cascade' }),
    body: text('body').notNull(),
    createdAt: timestamp('created_at', { withTimezone: true }).defaultNow().notNull(),
  },
  (t) => ({
    orgIdx: index('idx_task_comments_org').on(t.organizationId, t.taskId),
  }),
)

export const taskAttachments = pgTable(
  'task_attachments',
  {
    id: uuid('id').defaultRandom().primaryKey(),
    organizationId: uuid('organization_id')
      .notNull()
      .references(() => organizations.id, { onDelete: 'cascade' }),
    taskId: uuid('task_id')
      .notNull()
      .references(() => tasks.id, { onDelete: 'cascade' }),
    uploadedBy: uuid('uploaded_by')
      .notNull()
      .references(() => profiles.id, { onDelete: 'cascade' }),
    fileName: text('file_name').notNull(),
    // Supabase Storage object path (tenant-prefixed); downloads use signed URLs.
    storagePath: text('storage_path').notNull(),
    contentType: text('content_type').notNull(),
    sizeBytes: integer('size_bytes').notNull(),
    createdAt: timestamp('created_at', { withTimezone: true }).defaultNow().notNull(),
  },
  (t) => ({
    orgIdx: index('idx_task_attachments_org').on(t.organizationId, t.taskId),
  }),
)

export const timeEntries = pgTable(
  'time_entries',
  {
    id: uuid('id').defaultRandom().primaryKey(),
    organizationId: uuid('organization_id')
      .notNull()
      .references(() => organizations.id, { onDelete: 'cascade' }),
    taskId: uuid('task_id')
      .notNull()
      .references(() => tasks.id, { onDelete: 'cascade' }),
    userId: uuid('user_id')
      .notNull()
      .references(() => profiles.id, { onDelete: 'cascade' }),
    // Null minutes + non-null startedAt = a running timer.
    minutes: integer('minutes'),
    billable: boolean('billable').notNull().default(true),
    description: text('description'),
    source: timeEntrySource('source').notNull(),
    startedAt: timestamp('started_at', { withTimezone: true }),
    endedAt: timestamp('ended_at', { withTimezone: true }),
    createdAt: timestamp('created_at', { withTimezone: true }).defaultNow().notNull(),
  },
  (t) => ({
    orgIdx: index('idx_time_entries_org').on(t.organizationId, t.taskId),
    userIdx: index('idx_time_entries_user').on(t.organizationId, t.userId),
    // At most one running timer (minutes IS NULL) per user per task.
    oneRunning: unique('uniq_running_timer').on(t.userId, t.taskId, t.minutes),
  }),
)
```

> Note on `uniq_running_timer`: Postgres treats NULLs as distinct in a normal unique constraint, which would NOT prevent two running timers. The hard guarantee is added as a **partial unique index** in the RLS migration (Task 3). This Drizzle constraint is a no-op placeholder kept for type/index visibility; the real enforcement is the partial index. (We deliberately do not rely on it.)

Correction: remove the `oneRunning` line to avoid a misleading constraint. Replace the `timeEntries` index block `(t) => ({ ... })` with exactly:


```ts
const DEFAULT_COLUMNS = [
  { name: 'Intake', isIntake: true },
  { name: 'To Do', isIntake: false },
  { name: 'In Progress', isIntake: false },
  { name: 'In Review', isIntake: false },
  { name: 'Done', isIntake: false },
] as const

async function ensureBoard(organizationId: string, name: string, isOps: boolean): Promise<string> {
  const existing = await db
    .select()
    .from(schema.boards)
    .where(and(eq(schema.boards.organizationId, organizationId), eq(schema.boards.name, name)))
  if (existing[0]) return existing[0].id
  const [row] = await db
    .insert(schema.boards)
    .values({ organizationId, name, isOps })
    .returning()
  return row!.id
}

async function ensureColumns(organizationId: string, boardId: string): Promise<Record<string, string>> {
  const out: Record<string, string> = {}
  for (let i = 0; i < DEFAULT_COLUMNS.length; i++) {
    const col = DEFAULT_COLUMNS[i]!
    const existing = await db
      .select()
      .from(schema.boardColumns)
      .where(and(eq(schema.boardColumns.boardId, boardId), eq(schema.boardColumns.name, col.name)))
    if (existing[0]) {
      out[col.name] = existing[0].id
      continue
    }
    const [row] = await db
      .insert(schema.boardColumns)
      .values({ organizationId, boardId, name: col.name, position: i, isIntake: col.isIntake })
      .returning()
    out[col.name] = row!.id
  }
  return out
}

async function ensureTask(
  organizationId: string,
  boardId: string,
  columnId: string,
  title: string,
  origin: 'agency' | 'client',
  createdBy: string,
  position: number,
  assigneeId: string | null,
): Promise<string> {
  const existing = await db
    .select()
    .from(schema.tasks)
    .where(and(eq(schema.tasks.boardId, boardId), eq(schema.tasks.title, title)))
  if (existing[0]) return existing[0].id
  const [row] = await db
    .insert(schema.tasks)
    .values({ organizationId, boardId, columnId, title, origin, createdBy, position, assigneeId })
    .returning()
  return row!.id
}
```

```ts
export type ReorderInput = {
  taskId: string
  sourceColumnId: string
  targetColumnId: string
  /** Current ordering of task ids in the source column (including taskId). */
  sourceOrder: string[]
  /** Current ordering of task ids in the target column (excluding taskId if same column). */
  targetOrder: string[]
  /** Desired index of taskId within the target column. */
  newIndex: number
}

export type PositionUpdate = { taskId: string; columnId: string; position: number }

export type ReorderResult = { updates: PositionUpdate[] }

/**
 * Pure function: given the dragged task and the source/target column orderings,
 * compute the minimal dense (0..n) position rewrites for the affected column(s).
 * Same-column and cross-column moves are both handled.
 */
export function computeReorder(input: ReorderInput): ReorderResult {
  const { taskId, sourceColumnId, targetColumnId, sourceOrder, targetOrder, newIndex } = input
  const updates: PositionUpdate[] = []

  if (sourceColumnId === targetColumnId) {
    const without = sourceOrder.filter((id) => id !== taskId)
    const index = clamp(newIndex, 0, without.length)
    const next = [...without.slice(0, index), taskId, ...without.slice(index)]
    next.forEach((id, i) => updates.push({ taskId: id, columnId: targetColumnId, position: i }))
    return { updates }
  }

  // Cross-column: remove from source, insert into target.
  const sourceWithout = sourceOrder.filter((id) => id !== taskId)
  sourceWithout.forEach((id, i) => updates.push({ taskId: id, columnId: sourceColumnId, position: i }))

  const targetWithout = targetOrder.filter((id) => id !== taskId)
  const index = clamp(newIndex, 0, targetWithout.length)
  const targetNext = [...targetWithout.slice(0, index), taskId, ...targetWithout.slice(index)]
  targetNext.forEach((id, i) => updates.push({ taskId: id, columnId: targetColumnId, position: i }))

  return { updates }
}

function clamp(n: number, min: number, max: number): number {
  return Math.max(min, Math.min(max, n))
}
```

```ts
import type { AppRole, TaskOrigin } from '@/db/types'

export type Actor = { role: AppRole | null; userId: string }
export type TaskFacts = { origin: TaskOrigin; createdBy: string }

export function isStaffRole(role: AppRole | null): boolean {
  return role === 'founder' || role === 'team'
}

/**

 * any task; a client may mutate only client-origin tasks they created.
 * RLS is the hard gate — this exists for fast UI checks and clear 403s.

export function canMutateTask(actor: Actor, task: TaskFacts): boolean {
  if (isStaffRole(actor.role)) return true
  return task.origin === 'client' && task.createdBy === actor.userId
}

/** Reassigning (changing assignee) is staff-only. */
export function canReassignTask(actor: Actor): boolean {
  return isStaffRole(actor.role)
}
```

```ts
export async function updateTask(
  taskId: string,
  patch: { title?: string; description?: string | null; priority?: 'low' | 'medium' | 'high' | 'urgent'; dueDate?: string | null; assigneeId?: string | null },
) {
  const session = await requireSession()
  const userId = await assertAuthenticated()

  const [task] = await db.select().from(tasks).where(eq(tasks.id, taskId))

  if (!canMutateTask({ role: session.role, userId }, { origin: task.origin, createdBy: task.createdBy })) {
    throw new Error('Not allowed to edit this task')
  }

  const next: Partial<typeof tasks.$inferInsert> = {}
  if (patch.title !== undefined) next.title = patch.title

export async function reorderTask(input: {
  taskId: string
  sourceColumnId: string
  targetColumnId: string
  sourceOrder: string[]
  targetOrder: string[]
  newIndex: number
}) {
  const session = await requireSession()
  const userId = await assertAuthenticated()

  const [task] = await db.select().from(tasks).where(eq(tasks.id, input.taskId))
  if (!task) throw new Error('Task not found')
  if (!canMutateTask({ role: session.role, userId }, { origin: task.origin, createdBy: task.createdBy })) {
    throw new Error('Not allowed to move this task')

  const runningRows = await db
    .select()
    .from(timeEntries)
    .where(and(eq(timeEntries.taskId, taskId), eq(timeEntries.userId, userId)))
  const open = runningRows.find((r) => r.minutes === null && r.startedAt !== null)
  if (!open) throw new Error('No running timer')

  const endedAt = new Date()
  const minutes = Math.max(1, Math.round((endedAt.getTime() - open.startedAt!.getTime()) / 60000))
  await db
    .update(timeEntries)
    .set({ minutes, endedAt })
    .where(eq(timeEntries.id, open.id))
  // `running` referenced to satisfy lint when only the find result is used.
  void running
  revalidatePath(`/tasks/${taskId}`)
}

export async function addManualTime(input: {

  billable: boolean
  description?: string

  const session = await requireSession()

    taskId: input.taskId,

    billable: input.billable,
    description: input.description ?? null,
    source: 'manual',
  })
```

```tsx
'use client'

import { useState } from 'react'
import {
  DndContext,
  PointerSensor,
  useSensor,
  useSensors,
  type DragEndEvent,
} from '@dnd-kit/core'
import { SortableContext, verticalListSortingStrategy } from '@dnd-kit/sortable'
import { TaskCard, type CardTask } from './task-card'
import { reorderTask } from './actions'

type Column = { id: string; name: string; isIntake: boolean }

export function BoardView({
  boardId,
  columns,
  tasks,
}: {
  boardId: string
  columns: Column[]
  tasks: CardTask[]
}) {
  const [items, setItems] = useState<CardTask[]>(tasks)
  const sensors = useSensors(useSensor(PointerSensor, { activationConstraint: { distance: 6 } }))

  function tasksFor(columnId: string): CardTask[] {
    return items
      .filter((t) => t.columnId === columnId)
      .sort((a, b) => a.position - b.position)
  }

  async function onDragEnd(event: DragEndEvent) {
    const { active, over } = event
    if (!over) return
    const activeId = String(active.id)
    const activeTask = items.find((t) => t.id === activeId)
    if (!activeTask) return

    // Determine the target column: dropping over a card → that card's column;
    // dropping over a column container → the column id directly.
    const overId = String(over.id)
    const overTask = items.find((t) => t.id === overId)
    const targetColumnId = overTask ? overTask.columnId : overId
    if (!columns.some((c) => c.id === targetColumnId)) return

    const sourceColumnId = activeTask.columnId
    const sourceOrder = tasksFor(sourceColumnId).map((t) => t.id)
    const targetOrder = tasksFor(targetColumnId).map((t) => t.id)
    const newIndex = overTask
      ? targetOrder.indexOf(overTask.id)
      : targetOrder.length

    // Optimistic update.
    const without = items.filter((t) => t.id !== activeId)
    const reinserted: CardTask = { ...activeTask, columnId: targetColumnId }
    const targetIds = [
      ...targetOrder.filter((id) => id !== activeId).slice(0, newIndex),
      activeId,
      ...targetOrder.filter((id) => id !== activeId).slice(newIndex),
    ]
    const next = without.concat(reinserted).map((t) => {
      if (t.columnId === targetColumnId) {
        return { ...t, position: targetIds.indexOf(t.id) }
      }
      return t
    })
    setItems(next)

    // Server reconciliation (RLS-enforced); on failure, revert.
    try {
      await reorderTask({
        taskId: activeId,
        sourceColumnId,
        targetColumnId,
        sourceOrder,
        targetOrder,
        newIndex,
      })
    } catch {
      setItems(tasks)
    }
  }

  return (
    <DndContext sensors={sensors} onDragEnd={onDragEnd}>
      <div className="flex gap-4 overflow-x-auto">
        {columns.map((col) => {
          const colTasks = tasksFor(col.id)
          return (
            <div key={col.id} id={col.id} className="w-72 shrink-0 rounded-lg bg-muted/40 p-3">
              <h2 className="mb-3 flex items-center justify-between text-sm font-semibold">
                {col.name}
                {col.isIntake && <span className="text-xs text-muted-foreground">intake</span>}
              </h2>
              <SortableContext items={colTasks.map((t) => t.id)} strategy={verticalListSortingStrategy}>
                <div className="flex min-h-8 flex-col gap-2" data-column={col.id}>
                  {colTasks.map((t) => (
                    <TaskCard key={t.id} task={t} />
                  ))}
                </div>
              </SortableContext>
            </div>
          )
        })}
      </div>
    </DndContext>
  )
}
```

```ts
'use server'

import { revalidatePath } from 'next/cache'
import { asc, eq } from 'drizzle-orm'
import { db } from '@/db'
import { boards, boardColumns, tasks, taskComments } from '@/db/schema'
import { getSession, isStaff } from '@/lib/auth'

async function requireClient() {
  const session = await getSession()
  if (!session) throw new Error('Not authenticated')
  if (isStaff(session.role)) throw new Error('Use the internal app')
  if (!session.orgId) throw new Error('No organization on session')
  return session
}

export async function clientCreateTask(formData: FormData) {
  const session = await requireClient()
  const title = String(formData.get('title') ?? '').trim()
  if (!title) throw new Error('Title is required')

  // The client's own board for their org.
  const [board] = await db
    .select()
    .from(boards)
    .where(eq(boards.organizationId, session.orgId!))
    .orderBy(asc(boards.createdAt))
  if (!board) throw new Error('No board for your organization yet')

  // Intake column receives client-created tasks.
  const cols = await db
    .select()
    .from(boardColumns)
    .where(eq(boardColumns.boardId, board.id))
    .orderBy(asc(boardColumns.position))
  const intake = cols.find((c) => c.isIntake) ?? cols[0]
  if (!intake) throw new Error('Board has no columns')

  const existing = await db
    .select({ id: tasks.id })
    .from(tasks)
    .where(eq(tasks.columnId, intake.id))
  const nextPos = existing.length

  // origin='client', created_by=self → satisfies the RLS tasks_insert policy.
  await db.insert(tasks).values({
    organizationId: session.orgId!,
    boardId: board.id,
    columnId: intake.id,
    title,
    origin: 'client',
    createdBy: session.userId,
    position: nextPos,
  })
  revalidatePath('/tasks')
}

export async function clientAddComment(taskId: string, formData: FormData) {
  const session = await requireClient()
  const body = String(formData.get('body') ?? '').trim()
  if (!body) throw new Error('Comment cannot be empty')

  const [task] = await db.select().from(tasks).where(eq(tasks.id, taskId))
  if (!task) throw new Error('Task not found')

  // RLS comments_insert allows any task on the client's own board.
  await db.insert(taskComments).values({
    organizationId: task.organizationId,
    taskId,
    authorId: session.userId,
    body,
  })
  revalidatePath('/tasks')
}
```
